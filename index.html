Make a query to DEV.to in a static site using StepZen

⇦ StepZen's custom @rest directive operates behind the scenes in the sdl to retrieve dynamic information.

StepZen executes this query when you hit 'submit'. ⇨

It hits a pre-deployed StepZen endpoint with a live connection to the DEV.to API. 

The following information is returned from the StepZen endpoint ⇩

        query MyQuery {
          devto_getArticles(
            username: , 
            per_page: 3, 
            page: 1) {
            title
            canonical_url
            user {
              name
              twitter_username
              username
              website_url
            }
          }
        }
